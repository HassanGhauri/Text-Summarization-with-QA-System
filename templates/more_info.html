Project "TXT-SUM WITH Q/A"


"TXT-SUM WITH Q/A" ("Text Summarization with QA.") project is a website which allows a user to upload a text document

  • get it summarized or
  • get answers to the user's questions based on the summarized text.


Text is summarized following extractive summarization approach using distilbart-cnn-12-6 (distilbart-cnn-12-6) and question answering is implemented using a pre-trained machine learning model BERT.

Frontend: HTML/CSS-Bootstrap/JS.
Backend: Python/Flask.
